## About the Course

This course takes you from zero programming knowledge to being able to solve real Data Structures and Algorithms problems. We will move step by step — no rush.

**What you will learn (in order):**
1. **Python Fundamentals** — variables, operators, control flow, functions, file handling
2. **Object-Oriented Programming (OOPs)** — classes, objects, inheritance, polymorphism
3. **Data Structures & Algorithms (DSA)** — arrays, linked lists, stacks, queues, trees, sorting, searching

By the end, you will be confident writing clean Python code and solving standard DSA problems — a skill directly useful for placements and internships.

## How Assignments Work

1. Every week after class, an assignment link will be shared with you (via WhatsApp group).
2. Click the link — GitHub will create a **private repository** for you automatically.
3. Your repo will already contain the questions in `README.md` and empty `.py` files for each question.
4. Write your solutions in the `.py` files and **commit** (save) them before the deadline.
5. You do not need to do anything else — I can see your submission directly.

**Deadline:** Each assignment is due before the next weekly class.

## Setting Up Your Computer

You need two things installed:

### 1. Python
- Download from: https://www.python.org/downloads/
- During installation, check **"Add Python to PATH"**
- Verify: open a terminal and run `python --version`

### 2. VS Code (recommended editor)
- Download from: https://code.visualstudio.com/
- Install the **Python extension** (search "Python" in the Extensions tab)

### 3. Git
- Download from: https://git-scm.com/downloads
- This is needed to submit assignments on GitHub

### 4. GitHub Account
- Sign up at: https://github.com
- Use your college email if possible
- When you receive the assignment invite link, click it — GitHub Classroom will add you automatically
